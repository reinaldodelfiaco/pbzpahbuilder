# i18n/

Pasta de internacionalização (Qt Linguist).

- `pbzpa_pt_BR.ts` — fonte das traduções (a gerar com `pylupdate5`).
- `pbzpa_pt_BR.qm` — compilado das traduções (a gerar com `lrelease`).

A interface principal já está em pt-BR; o esqueleto i18n permite
distribuir versões em inglês/espanhol no futuro.
